{{- if .Values.zitadel.bundled }}

── Zitadel Initialization ──

The zitadel-init Job will run automatically and:
  1. Create the StackWeaver organisation, project, and OIDC apps
  2. Create service users and PATs
  3. Configure webhooks and IdP sync
  4. Write all credentials directly into the Zitadel secret
  5. Trigger rolling restarts of the API, frontend, and login-ui

No manual steps are required. To monitor progress:
  kubectl logs -f job/{{ include "stackweaver.fullname" . }}-zitadel-init
{{- end }}
